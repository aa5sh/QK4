About

QK4 was created to fill a gap in the Elecraft ecosystem — a native, purpose-built remote control application that runs on any platform. No emulators, no workarounds, no compromises. Full spectrum and waterfall display, dual VFO control, audio streaming, and CW/voice operation over your local network.

Requirements

macOS macOS 14 (Sonoma) or later
Apple Silicon (M1/M2/M3/M4)
Windows Windows 11, x64
VC++ Redistributable 2019+
Pi Pi 4 or 5, ARM64
Debian Trixie

Getting Started

macOS Open the DMG and drag to Applications.
Signed and notarized — no Gatekeeper warnings.
Windows Extract the ZIP and run QK4.exe.
Install VC++ Redistributable if prompted.
Pi Extract the tarball and run ./QK4/run.sh.
First run requires sudo to install a udev rule for KPOD USB access. After that, sudo is not needed.
